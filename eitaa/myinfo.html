دریافت اطلاعات اکانت


اگه بخوایم اطلاعات توکن رو دزیافت کنید میتونید از کد زیر استفاده کنید 
from Eitaabot import Bot
bot = Bot("TOKEN")
info = bot.get_my_info()
print(info)
خب، خط اول و دوم که براتون آشناست و میدونین باید چکار کنین اما خط سوم میایم با استفاده از متن get_my_info استفاده میکنیم و طلاعات رو دریافت میکینم و میریزیم داخل متقیر info و در نهایت متقیر info رو پرینت میکنیم
